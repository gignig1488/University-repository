Task 1 – Exploring Data Types in R

1.1 Atomic Types

Let’s define attributes of our fictional planet Aqualis.

# Numeric values
temperature <- 24.5
gravity <- 9.8
ocean_depth <- 3500.2
radiation_level <- 0.87

# Integer values
number_of_moons <- 2L
cities_count <- 8L
research_stations <- 3L

# Logical values
has_water <- TRUE
is_inhabited <- TRUE

# Character values
planet_name <- "Aqualis"
region <- "Hydros Sector"
category <- "Ocean Planet"

# Describe them in a sentence
cat(
  planet_name, "is located in", region, 
  "and is classified as an", category,
  "with", number_of_moons, "moons and average temperature", temperature, "°C.\n"
)
Aqualis is located in Hydros Sector and is classified as an Ocean Planet with 2 moons and average temperature 24.5 °C.

# Numeric + integer vector
numeric_vector <- c(temperature, gravity, ocean_depth, radiation_level,

1.3 Matrix

# Create 3x3 matrix
set.seed(123)
planet_matrix <- matrix(sample(-50:50, 9, replace = TRUE), nrow = 3, ncol = 3)

# Name rows and columns
rownames(planet_matrix) <- c("Ocean", "Land", "Atmosphere")
colnames(planet_matrix) <- c("Pressure", "Temperature", "Density")
